# Hands-free — running the whole business by voice and taps

Typing is costly for Johnny, so the neighborhood is built the other way around: **his side
of everything is a spoken word or a single tap.** The Hands Covenant
(`agents/mr-rogers.md`) binds every agent to this. This page is the practical how.

## Set up dictation once (no typing required to do it)
- **iPhone/Android:** the 🎤 mic key on the keyboard — tap it, talk, done. Works in
  GitHub, Telegram, email, everywhere.
- **Mac:** press **fn fn** (the globe/fn key twice) → dictate into any box. For fuller
  control (open apps, click by voice): System Settings → Accessibility → **Voice
  Control** → on. (Saying "Show numbers" overlays clickable numbers on everything.)

## The doors, ranked for hands-free
1. **🎤 Talk to a Claude Code session** — literally just speak (dictation into the chat).
   Mr Rogers does 100% of the keyboard work. *The best door.*
2. **📱 Telegram** (once connected): voice-type any message; `/task fix the about page`.
3. **🌸 HQ / Desk buttons** — every prefilled note is **sendable exactly as-is**; Rogers
   asks ONE question on the thread and you dictate the answer.
4. **📧 Email bridge** (once connected): dictate an email to rogers@braveplumhealing.com.

## Reviews & approvals = one tap
PRs: open the link → tap **Merge** (GitHub mobile app makes this a thumb-tap). Yes/no
questions from Rogers: reply with a dictated "yes." That's the whole human gate.

## The remaining "Johnny-only" tasks, hands-free style
These stay yours by *safety design* (credentials, money, consent) — but your part is
clicks and voice only. For each: **open the page, tell Rogers "walk me," and he narrates
one click at a time** (concierge pattern — like the Stripe branding and token walks).
| Task | Your actual hands-on part |
|---|---|
| www DNS record | click Add Record; dictate/select the values Rogers reads to you |
| Passkeys/2FA | click "Add passkey" → touch the sensor (that's the point of passkeys — no typing!) |
| GoatCounter signup | clicks + one dictated code back to Rogers |
| Old $228 Stripe link off | Stripe app → Payment Links → tap → Deactivate |
| Registrar auto-renew/lock | two toggles |
| Telegram/email bridges | Rogers types every command locally; you click Allow in the browser and paste-with-one-key (⌘V) anything secret |

**Passkeys are your best friend** — they replace typed passwords with a fingerprint/face
tap. DURABILITY item 1 just got easier, not harder.

## The standing promise
If any agent ever asks you to type something longer than a dictated sentence, that's a
bug in the neighborhood — say "hands" and it becomes their keyboard work.
