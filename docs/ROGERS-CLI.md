# rogers - Johnny's one-word control panel (runbook)
Type `rogers` in Terminal. Menu: 1 status / 2 waiting / 3 launch order /
4 merge PR (shows one-sentence test, asks y/N) / 5 ask anything (typos fine) / q.
Safety: read-only except launch (go-ahead comment) and merge (one PR, confirmed).
Cannot delete, pay, or change settings. Installed: scripts/rogers -> ~/bin/rogers.
